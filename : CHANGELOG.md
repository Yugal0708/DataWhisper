# Changelog

All notable changes to DataWhisper will be documented here.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-05-01

### Added
- CSV file upload with drag-and-drop support
- Automated EDA: summary stats, missing value heatmaps, correlation matrices
- Distribution and count plots via Matplotlib & Seaborn
- AI-powered insights using LangChain + Groq/OpenAI
- Interactive chat interface with Pandas DataFrame Agent
- Data preprocessing recommendations via LLM
- HTML report export with visualizations and AI insights
- Sample Titanic dataset for quick exploration
- Streamlit OAuth and Authenticator-based login
- Multi-step navigation sidebar
- Auto AI Executive Summary on file upload
- Explain chart feature for individual visualizations
- AI task modes: Anomaly Detection, Deep Correlations, Full Summary
